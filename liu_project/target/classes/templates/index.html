About The System

每一样东西都有它存在的意义与价值，人也一样即使再渺小。

能充分利用资源的就是强者，倘若没有那么就是缺少一个途径。该平台致力于信息的发布与咨询，校园闲置物品的再使用，物品的等价交换，丢失物品的寻找，以及人力资源效益的最大化

你手中所认为的废品可能对他人来说有着巨大的价值。你心急如焚想要找到的有着特别意义的失物可能就在某人的眼前与手中只是他不知道而已。你不知道的事和走的弯路也许别人早就经历过了，而你与避过这些只差前人的一句忠告，你迫切的想要得到的帮助对于他人来说可能就是举手之劳，可惜他听不到也看不见。

一个人的圈子很小很小能力也极其有限，那我们就更需要一根绳子将千万个你我他串联起来组成一个强大的集体。这样既有利于我也有利于你。 社会是极其残酷的，但是如果能和老师，学哥学姐，学弟学妹们一起砥砺前行的话前景一定很美妙。

已经走入社会的你们，将要步入社会的我们，还有在校园的他们 大家都属于弱势群体，所以我们更应该抱团取暖。你以前所帮助过的懵懂学弟在不久的将来可能就是你的得力干将，左膀右臂。 也可能在你事业生死一线的时候助你起死回生一飞冲天，也有可能多年后他成为了你的上司。 世界之大无奇不有。
